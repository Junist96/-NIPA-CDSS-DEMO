Description

University Hospital klinikum Bonn에서 제공하는 뇌파 데이터는 500명의 피험자를 대상으로 128채널의 앰프 시스템을 이용해 23초 가량 뇌파 데이터를 측정하고 기록하였다. 기록 된 뇌파 데이터는 아날로그 타입이였으나, 오픈데이터로 공개하기 위해 12 비트의 디지털 데이터로 변환 과정을 거친 뒤 배포 되었다.

500명의 뇌파 데이터는 피험자의 상태에 따라 A, B, C, D, E로 총 5개의 분류로 나누었으며, 각각의 분류 폴더에는 100명의 피험자로부터 23초 동안 기록한 뇌파 데이터를 구분하여 배포하였다. 피험자의 상태를 분류하는 기준은 다음과 같다. A와 B 분류는 건강한 피험자로부터 측정한 뇌파이며, A는 눈을 뜬 상태로 측정한 뇌파이며, B는 눈을 감은 상태로 측정한 뇌파이다. C와 D는 뇌에 종양이 발견된 피험자로부터 측정한 뇌파이며, C는 종양이 위치하지 않은 영역에서 측정한 뇌파이며, D는 종양이 위치한 영역에서 측정한 뇌파이다. E는 간질에 의해 발작을 일으키는 뇌파를 측정한 뇌파이다.

데이터 셋은 로체스터 공과대학(RIT; Rochester Institute of Technology)에서 원본 데이터를 전처리 및 재구성하여 제공하는 데이터 셋이다. 데이터 셋에는 다양한 부위에서 측정한 뇌파 데이터 값을 X1에서 X178까지의 속성에 저장하였으며, 데이터의 레이블로 피험자의 상태를 나타내는 값을 y 속성에 저장하였다. 각각의 속성에는 500명의 피험자가 23초 동안 측정한 11,500개의 데이터가 저장되어 있으며, 총 2,058,500개의 데이터 포인트 구성되어 있다.
